Log 文檔分析  on linux/local

功能 : 參考參數設定過濾文件，並解析成指定格式後輸出

目前 on linux 已完成，local部分有待實作
=============================================================================
專案建置 : 

1. 至 Project Properties -> Project Facets -> Convert to faceted form...
2. java 請打勾，點擊 OK
3. java build path -> Libraries -> Add JARs -> 專案目錄下 lib 全選
4. /src/config.properties 指定參數
5. 範例文檔與輸出結果請參照 /log_example
